Key Features

🔐 Secure Auth

Institutional email + 6-digit PIN. Hashed passwords, httpOnly cookies, session management.

✓ Attendance

TOTP-style rotating 6-digit codes. Changes every 45 seconds. Server-side validation.

📅 Events

Full CRUD for admins. Students see published events with descriptions and linked attendance.

🔔 Notifications

OneSignal Web Push. Target all students, cohorts, or custom audiences. Deep linking to events.

👥 Cohort Privacy

Row-Level Security (RLS) enforces cohort isolation at database level. No data leaks.

📱 PWA Ready

Install on any device. Works offline. Push notifications on mobile and desktop.

Tech Stack

Frontend

Next.js 16 (Turbopack), React 19, Tailwind CSS v4

Backend

Supabase PostgreSQL, Server Actions, API Routes

Auth & Security

Supabase Auth, Row-Level Security, Session middleware

Notifications

OneSignal Web Push, Vercel Cron for scheduling

Data Import

exceljs for Excel/CSV parsing, batch processing for 1000+ rows

Deployment

Vercel (auto-deploy on git push), environment variables
